LaTeX academic paper template based on IEEE standards (Normas IEEE em LaTeX), using the **IEEEtran** document class for conference and journal submissions.

## Features

- Pre-configured IEEE conference paper template
- Based on the official [IEEEtran](https://www.ieee.org/conferences/publishing/templates.html) document class
- Ready-to-use structure for academic papers

## Installation

1. Clone the repository:

```bash
git clone https://github.com/rios0rios0/acs-tex.git
```

2. Open the `.tex` files in your preferred LaTeX editor (e.g., Overleaf, TeXstudio, VS Code with LaTeX Workshop).

## Usage

Edit the `document.tex` file inside the article directory to write your paper. Compile with `pdflatex` or your preferred LaTeX compiler:

```bash
pdflatex document.tex
```
